## ✨ Features

### 🔐 Authentication
- Simple & clean login screen
- Animated logo with fade/slide transitions
- Pre-filled demo credentials
- Form validation with error messages
- Loading state with progress indicator

### 📊 Market Watch
- Real-time price updates (mocked every 5 seconds)
- Color-coded price change indicators
  - 🟢 Green = Price Going Up
  - 🔴 Red  = Price Going Down
- Mini sparkline charts on each stock card
- Search stocks by name or symbol
- Filter by: All, Gainers, Losers, Favorites, Tech, Finance
- Market summary bar (Total, Gainers, Losers count)
- Shimmer loading animation
- Smooth scroll with scroll-to-top button

### 📈 Stock Detail View
- Live animated price display
- % change with direction arrow
- Interactive line chart (intraday 1D trend)
- Time range selector: 1D, 1W, 1M, 3M, 1Y
- Key Statistics grid:
  - Open, Close, High, Low
  - Volume, Market Cap
- About section with sector info
- BUY / SELL action buttons
- Trade confirmation dialog

### ⚙️ Settings Modal
- 🌙 Dark / Light mode toggle
- ⏱ Refresh interval: 3s / 5s / 10s
- 💱 Currency display: USD, EUR, GBP, JPY, INR
- App version info
- Floating Action Button access

### 📋 Trade History
- Portfolio summary card with total volume
- Tab filter: All / Buy / Sell
- Status filter: Completed, Pending, Cancelled, Failed
- Each trade shows:
  - Symbol & Stock Name
  - Order Type (BUY/SELL) badge
  - Quantity & Price
  - Total Value
  - Timestamp
  - Order ID
  - Status with color coding

### 🔔 Notifications
- Push notification panel
- Price alert notifications
- Market open/close alerts

### 🎨 UI/UX
- Dark theme by default
- Smooth page transitions (slide + fade)
- AnimatedSwitcher for price updates
- Bouncing scroll physics
- Responsive layout
- Custom sparkline charts
- Gradient backgrounds
